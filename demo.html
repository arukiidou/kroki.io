Kroki

Convert plain text diagrams to images !

Kroki provides a unified API with support for AsciiToSVG, BlockDiag (BlockDiag, SeqDiag, ActDiag, NwDiag), Ditaa, Erd, GraphViz, PlantUML, SvgBob and UMLet... and more to come!

Examples

Here's a list of example to get you inspired.

Block diagram

BlockDiag

blockdiag {
  blockdiag -> generates -> "block-diagrams";
  blockdiag -> is -> "very easy!";

  blockdiag [color = "greenyellow"];
  "block-diagrams" [color = "pink"];
  "very easy!" [color = "orange"];
}
