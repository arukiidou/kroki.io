Kroki

Convert plain text diagrams to images !

Kroki provides a unified API with support for AsciiToSVG, BlockDiag (BlockDiag, SeqDiag, ActDiag, NwDiag), Ditaa, Erd, GraphViz, PlantUML, SvgBob and UMLet... and more to come!

Usage

Kroki uses a simple algorithm (deflate + base64) to encode your diagram in the URL:

GET /plantuml/png/SyfFKj2rKt3CoKnELR1Io4ZDoSa70000

You can also call Kroki with POST:

POST /
{
  "digram_source": "Bob -> Alice : hello",
  "diagram_type": "plantuml",
  "output_format": "png"
}

In this case, you don't need to encode your diagram.
